SVG Icon

2018-01-01
icon, svg, svgicon, web

Scalable Vector Graphics

확장가능하므로, 어떤 해상도에서든 선명한 이미지를 보여준다

크로스브라우징

  • PC에서는 SVG를 미지원 하는 경우가 많음
  • SVG 지원 환경과 미지원 환경을 분기 처리해서 제공해야 함
  • SVG의 사용을 선택할 경우 ﻿GIF와 SVG를 동시에 관리해야 함


용량

  • SVG 사용시 최대의 이슈는 SVG 용량을 줄이는 것에 있다. (GIF:22kb, SVG:144kb, 약 6배)
  • SVG 파일의 용량이 큰 이유
    • 텍스트의 경우 SVG로 표현시 용량이 클 수 밖에 없음
    • 반복되는 그래픽 요소에 대한 효율적인 접근이 파일 자체에서는 불가능(재사용 등의 이슈)
  • SVG 파일 크기를 줄이기 위한 노력
    • 시스템 폰트를 사용하고 그래픽 요소만 SVG로 사용
      • 3kb로 제일 용량이 작음
      • 시스템 폰트를 이용하므로 디자인 상의 검토가 필요함
    • 텍스트 SVG와 그래픽요소 SVG를 분리 하여 구성
      • 용량이 78kb 정도
      • 디자인과 동일한 퀄리티
      • 관리해야하는 SVG파일이 증가함
    • 언급한 내용외에도 다른 방법은 충분히 있을 수 있음
  • SVG 파일 크기를 줄이기 위한 노력중 효과가 크지 않은 것들
    • SVG 파일 내의 불 필요한 태그 제거 : 다소 효과가 느껴지지만 수동으로 제거해야함
    • SVG 파일 내의 공백 제거를 통한 최적화 : 효과는 크지 않으나 작업 난이도가 매우 낮음
    • 좌표의 값에 대한 소수점 제한을 통한 파일크기 경량화 : 그래픽 요소가 많은 경우 효과적 일 수 있으나, 이번 메뉴 작업과 같은 경우 큰 효과 없음
    • 위 세가지의 방법은 파일 경량화의 중요도에 따라 작업 진행을 결정하면 좋을 듯


SVG 데이터 소수점

  • 일러스트레이터 SVG 저장 옵션 중 SVG 데이터 수치의 소수점 자릿수 지정하는 옵션
  • SVG 데이터 정밀도에 밀접한 관련이 있음.
  • 값이 높을수록 시안과의 정밀도는 높아지지만 파일 용량은 증가
  • 기본값으로는 3, 즉 소수점 셋째짜리까지 SVG 데이터 값이 표시됨
  • 하지만, 파일 용량을 줄이는데에 큰 효과를 없었음
